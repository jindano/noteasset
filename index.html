단계 1 · Firebase 프로젝트 만들기
  • 1 console.firebase.google.com에 접속합니다.
  • 2 Google 계정으로 로그인한 뒤 프로젝트 추가 버튼을 클릭합니다.
  • 3 원하는 프로젝트 이름을 입력하고 계속을 클릭합니다. Google Analytics는 선택 사항입니다.
  • 4 프로젝트 생성 완료 후 계속을 클릭하면 프로젝트 대시보드로 이동합니다.
단계 2 · Realtime Database 만들기
  • 1 왼쪽 사이드 메뉴에서 빌드 → Realtime Database를 클릭합니다.
  • 2 데이터베이스 만들기 버튼을 클릭합니다.
  • 3 데이터베이스 위치를 선택합니다. 국내 사용 시 asia-southeast1 (싱가포르)를 권장합니다.
  • 4 다음을 클릭합니다. (보안 규칙은 다음 단계에서 별도 설정합니다.)
단계 3 · 보안 규칙 설정
  • 1 Realtime Database 화면에서 상단 규칙 탭을 클릭합니다.
  • 2 아래 규칙을 복사해 붙여넣은 뒤 게시 버튼을 클릭합니다.
⚠️ 아래 규칙은 누구나 읽기·쓰기가 가능합니다. 내부망이나 신뢰할 수 있는 환경에서만 사용하세요. 외부 공개 서비스라면 Firebase Authentication을 적용하고 인증된 사용자만 접근하도록 규칙을 변경해야 합니다.
{ "rules": { ".read": true, ".write": true } }
단계 4 · 웹 앱 등록 및 Config 확인
  • 1 상단 왼쪽 톱니바퀴(⚙️) 아이콘 → 프로젝트 설정을 클릭합니다.
  • 2 페이지 아래의 내 앱 섹션에서 웹 아이콘 </>을 클릭합니다.
  • 3 앱 닉네임을 입력하고 앱 등록을 클릭합니다. (Firebase Hosting은 선택 사항입니다.)
  • 4 화면에 표시된 firebaseConfig 객체의 각 값을 다음 단계에서 입력합니다.
단계 5 · Firebase Config 입력 및 저장
💡 저장 후 페이지를 새로고침하면 Firebase 동기화가 시작됩니다. 설정을 초기화하려면 빈 칸으로 저장하세요.
